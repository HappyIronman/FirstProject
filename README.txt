代码分为几个文件夹：
src/com.graduateProject文件夹下的内容为后台代码
web文件夹下的为前端页面代码和服务器配置文件。

后台：
src/com.graduateProject文件夹下

controller文件夹下的文件负责接受前端请求，并且返回相应数据
entity文件夹下的每个文件定义了一个对应的实体，包括（Customer客户，Finance财务数据，Model模具，Provider供应商，Raw_material原材料，User用户）
DAO文件夹下的文件声明了访问数据库的接口
mapper文件夹下的文件用于访问数据库，从数据库中获取数据，执行数据库访问SQL语句
service_inter文件夹定义的是数据处理函数的接口
service文件夹下实现了service_inter定义的数据处理函数接口，用来处理从数据库中获取的数据


前台：
web文件夹下

css文件夹存放css文件，负责页面的布局和美化
images文件夹存放显示在页面上的文件
js文件夹负责页面特效

web-inf文件夹下
pages文件夹为对应页面的主体html代码（每个文件开头有对应页面注释）
applicationContext是jdbc和mybatis数据库组件的配置代码
dispatcher-servlet配置扫描器
web.xml是服务器相应配置代码，包括分发过滤器等
